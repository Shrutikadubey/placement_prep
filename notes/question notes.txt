3-2. Running Sum of 1D Array
🧩 Pattern:

👉 Prefix Sum (VERY IMPORTANT ⭐)

💡 Idea (notes में ऐसे लिखना है):
Pattern: Prefix Sum

Idea:
- Current element = previous sum + current value
- nums[i] = nums[i] + nums[i-1]
- Build cumulative sum step by step
🧠 Recognition Trick:
If question asks:
- “running sum”
- “cumulative sum”
- “sum from 0 to i”
→ Think: Prefix Sum
